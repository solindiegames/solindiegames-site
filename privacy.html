UMPB Privacy Policy

Privacy Policy for Ultra Mega Planet Battles (UMPB)

Last updated: 5 May 2026
Developer: Solindie Games

1. Overview

Ultra Mega Planet Battles ("the Game") is developed and published by Solindie Games.

This Privacy Policy explains how the Game handles user data on Android devices.

UMPB is a single-player offline game. It does not collect, store, transmit, or share any personal or sensitive user data.

2. Information We Do Not Collect

The Game does not collect or process any of the following:

  • Personal information (name, email, phone number, address)
  • Device identifiers (Android ID, advertising ID, IMEI, etc.)
  • Location data (GPS, network-based, or coarse location)
  • Financial or payment information
  • Contacts, photos, media, or files
  • Usage analytics or gameplay metrics
  • Crash logs or diagnostics
  • IP addresses or network information
  • Any data that can identify an individual user

The Game does not create user accounts and does not require login of any kind.

3. No Internet Connectivity Required

The Game is fully playable offline. It does not:

  • Connect to external servers
  • Sync data
  • Upload gameplay information
  • Communicate with any online services

The only optional online interaction is when the user chooses to tap a link directing them to the official Steam page for the PC version of the game.

4. No Third-Party Services or SDKs

The Game does not integrate any third-party services, including:

  • Advertising networks
  • Analytics providers
  • Social media SDKs
  • Cloud services
  • Third-party login systems

No third-party code collects or processes user data.

5. In-Game Promotion of the Steam Version

The Game includes optional, non-intrusive informational content that informs players about the availability of a separate paid version of UMPB on Steam.

This promotional content:

  • Does not track user behaviour
  • Does not personalise content
  • Does not collect data
  • Does not involve advertising networks
  • Is entirely user-initiated (only activated when the player taps the link)

Following the link opens the Steam store page in the user's browser or Steam app.

6. Children's Privacy

The Game does not collect any data from users of any age.

If the Game is accessed by children, no personal information is collected, stored, or shared.

7. Data Storage and Security

Because the Game does not collect or store any user data, no personal information is retained on servers or transmitted externally.

All gameplay progress is stored locally on the user's device and remains entirely under the user's control.

8. Changes to This Privacy Policy

If the Game is updated in the future to include features that require data collection or online services, this Privacy Policy will be updated accordingly.

Any changes will be posted on the official Solindie Games website.
